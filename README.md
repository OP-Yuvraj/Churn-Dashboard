📊 User Churn Analysis Dashboard

An interactive User Churn Analytics Dashboard built using Python and Streamlit to analyze customer behavior, engagement patterns, and churn insights.

🚀 Project Overview

This project focuses on analyzing user churn behavior using engagement and experience metrics.
The dashboard provides business-ready insights through KPIs, visualizations, filters, and downloadable reports.

The goal of this project is to simulate a real-world SaaS churn analytics system.

📊 Business Insights Extracted

Lower engagement often correlates with churn.

Rating inconsistency (high standard deviation) can indicate dissatisfaction.

Users with fewer active days and interactions show higher churn probability.
